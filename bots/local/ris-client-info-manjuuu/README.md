# ris-client-info-manjuuu クライアント概要

このクライアントは Discord サーバー上で動作する Python 製のBotです。

## 主な機能
- `/manjuuu ヘルスチェック` : Botの稼働確認メッセージを返します。
- `/manjuuu こんにちは` : 挨拶メッセージを返します。
- `/manjuuu ランダム文字列` : 12文字のランダムな英数字文字列を返します。
    - 5%の確率で、過去1週間分のチャットからランダムな一行を返します。
- `/manjuuu 575` : 5-7-5形式の投稿を受け付けるモードを開始します。
    - 実行後、同一ユーザー・同一チャンネルの発言を継続待機します。
    - `/manjuuu 退勤` でモードを終了します。
    - Janomeを使ったモーラ解析で5-7-5判定を行います。
    - 漢字を含む語のみモーラ数の揺れを ±1 まで許容します。
    - 不自然な句切れを減らすため、助詞境界優遇・活用途中分割抑制のルールを適用します。
    - 季語リスト（コード内定数）に一致した場合は「あっぱれ！」を返します。

## 技術構成
- discord.py
- Python 3.12
- Janome
- .envによるトークン管理

## 実装ファイル
- `main.py` : クライアント本体
- `services/randam_string_service.py` : ランダム文字列・チャット取得等のサービス

## 575機能の補足
- 5-7-5判定は文字数ではなくモーラ数（拍）ベースで判定します。
- 判定処理は `services/randam_string_service.py` の以下メソッドで実装しています。
    - `_analyze_mora`
    - `_split_575`
